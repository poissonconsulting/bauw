# House martin annual counts

The house martin (*Delichon urbica*) population annual counts from
Magden (a small village in Northern Switzerland) collected by Reto
Freuler from 1990 to 2009.

## Usage

``` r
hm
```

## Format

A data frame with 20 rows and 2 columns

## Details

The variables are as follows:

- `hm` the count (integer).

- `year` the year (integer).

## References

Kery M & Schaub M (2011) Bayesian Population Analysis using WinBUGS.

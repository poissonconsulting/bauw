# Showy lady's slipper capture-recapture data

The showy lady's slipper (*Cypripedium reginae*) capture-recapture data
collected by Kathy Gregg in Big Draft (West Virginia) from 1989 to 1999
from Kery & Schaub (2011 p.166).

## Usage

``` r
orchids
```

## Format

A data frame with 250 rows and 11 columns

## Details

The variables are as follows:

- `V1, V2, ..., V11` the state of the individual plant by year where 0 =
  undetected, 1 = vegetative and 2 = flowering.

## References

Kery M & Schaub M (2011) Bayesian Population Analysis using WinBUGS.
Academic Press.
(<https://www.vogelwarte.ch/en/research/population-biology/book-bpa/>)

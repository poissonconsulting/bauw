# Peregrine falcon breeding population data

The peregrine falcon (*Falco peregrinus*) population breeding in the
French Jura from 1964 to 2003.

## Usage

``` r
peregrine
```

## Format

A data frame with 40 rows and 4 columns

## Source

Kery & Schaub (2011 p.64-65) courtesy of R.-J. Monneret

## Details

The variables are as follows:

- `Year` the year (integer).

- `Pairs` the number of adult pairs (integer).

- `R.pairs` the number of reproductive pairs (integer).

- `Eyasses` the number of fledged young (integer).

## References

Kery M & Schaub M (2011) Bayesian Population Analysis using WinBUGS.
Academic Press.
(<https://www.vogelwarte.ch/en/research/population-biology/book-bpa/>)

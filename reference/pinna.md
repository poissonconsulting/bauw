# Pen shell detection data

The pen shell (*Pinna nobilis*) detection data from the Balearic Islands
in 2010.

## Usage

``` r
pinna
```

## Format

A data frame with 143 rows and 3 columns

## Source

Kery & Schaub (2011 p.166) courtesy of Iris Hendriks and colleagues

## Details

The variables are as follows:

- `d1` indicator for shell detected by first team.

- `d2` indicator for shell detected by second team.

- `width` shell width (cm).

## References

Kery M & Schaub M (2011) Bayesian Population Analysis using WinBUGS.
Academic Press.
(<https://www.vogelwarte.ch/en/research/population-biology/book-bpa/>)
